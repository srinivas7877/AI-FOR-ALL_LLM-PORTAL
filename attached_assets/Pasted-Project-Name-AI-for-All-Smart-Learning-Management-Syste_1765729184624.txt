Purpose of the System
The purpose of AI for All LMS is to provide a web-based learning platform where users can access courses, track progress, and interact with an AI-powered assistant that supports learning across all modules.

The system will allow:

Learners to enroll and study courses

Admins to manage content and users

AI to assist users throughout the platform

Scope of the Project In Scope
User Authentication (Login / Signup)

Course Management

AI Assistant integrated across the UI

Progress Tracking

Admin Dashboard

Fully responsive UI

Deployment-ready system

Out of Scope

Live video streaming

Payment gateway (optional future phase)

Native mobile apps (web-based only)

User Roles 3.1 Admin
Manage users

Create and manage courses

Control AI settings

3.2 Learner (User)

Register and login

Browse and enroll in courses

Access learning materials

Interact with AI assistant

Track learning progress
